What is Class in Ruby?

Classes are objects that are used to organize other objects and classes. Everything in Ruby is either an object or a construct that evaluates to an object. Every object is an instance of some class. Integers are instances of a class, and classes themselves are objects.

Let's create a class of food that will apply to all other types of food

Class Food (The name of the class (aka constants) is Food and must be capitalized!) @edible = "chew and swallow" def yummy (instance method: method defined in class and intended for use by all instances of this class. so any instance of this class can call them.) "Get in my Belly!"

candy = Food.new (this would be an instance of the class "Food") p candy.yummy
