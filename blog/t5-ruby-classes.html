Dear Fellow Beginners,

What is a "Class" in Ruby?

First of all, everything in Ruby is either an object or will yield an object. Classes are objects to be used to organize other objects and classes. These other objects are then instances of the class.

Analogy time: There's a class of wizards. When we created this class, we can define a method to tell us that "It's a wizard!" that applies to all wizards. Then we create an instance of such wizard and call it Hermione, therefore, Hermione is an instance of a wizard. We can put more methods when we create her but because she called her a wizard, we'll get alert that "It's a wizard!" We could also create Harry Potter and add him as an instance of a wizard. He'll also get an alert. Let's do it!

Example: Let's create the Wizard Class

To Create the Class:

class WizardHere   def initialize
  @alert = alert

Be sure to capitalize the first letter of the name. Then, "Camel" it by capitalizing the first letter of each word after it, rather than using underscore.

Define Instance Method of this Class:

def alert
  "It's a Wizard!"
end

No caps and use underscore please. Once you define the method inside the class, all instances of the class will share this method. Don't forget to put an "end" to close off the class!

Create and call Hermione and Harry as an instance of a Wizard!

hermione = Wizard.new

harry = Wizard.new

the full code looks like this:

class Wizard
  def initialize
  @alert = alert
end
 def alert
  p "There's a Wizard here!"
  end
end

Call it!

hermione = Wizard.new
harry = Wizard.new
