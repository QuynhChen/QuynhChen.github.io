Ruby Hashes vs. JavaScript Objects
Hash and Objects ==> Sisters from another mother? 11/2/14
Dear Fellow Beginners,

rb Hash and js Objects: SO similar, How to tell the difference?

They work in similar ways of keeping information that is related to each other. Let's take a look at syntax.

Ruby hash syntax

name_of_hash = {
  "name_of_key" => value1
.   "name_of another_key" => value2 }

JavaScript object syntax

var name_of_object = { name: "Terah", name: 32 }
