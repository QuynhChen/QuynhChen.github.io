What is Ruby inheritance?

Ruby inheritance is a relationship between two classes. One is the parent, the other is the child. The child class “inherits” any methods and constants from the parent class. The result is that the child class acquire the methods defined in the superclass.

class Mammals
  attr_accessor :hair
end

class Bears < Mammals
 attr_accessor :claws
end

Here, the class Bears would have it's own method called "claws" built into the class Bears. In addition, it would inherent all the class Mammals attributes. Here the attribute in Mammals is a method named "hair." So class Bear also has the method "hair." The reverse is not true: class Mammals cannot access the method "claws".

What is composition?

Traditional composition involves moving those features that we want to share into a class in their own right.
